POETRY PLEASE LINK GUIDE

This guide explains how to build direct links into Poetry Please at:
https://poetryplease.org


BASE URLS

Desktop app:
https://poetryplease.org/app

Auto-detect entry:
https://poetryplease.org/

Single-item embed:
https://poetryplease.org/embed

The auto-detect entry preserves link parameters, so both of these work:
https://poetryplease.org/app?type=EXC
https://poetryplease.org/?type=EXC


SUPPORTED PARAMETERS

item
Sends a user to a specific piece of content first.

type
Filters content by content type.

Special type alias:
type=video includes both hosted Poetry Please videos (VV) and YouTube-backed videos (YT).

catalog
Filters content by release catalog.

author
Filters content by author.

book
Filters content by book.

These parameters can be combined.


BEHAVIOR

If item is provided, the app tries to show that item first.

After loading the first item, the full app keeps serving more content that matches the active filters.

Existing in-app controls still work after landing from a link.

The URL updates as filters change in the full app.

The embed route uses the same filter parameters, but serves a single item at a time with a button to open the full app.


IMPORTANT NOTES

Parameter values should be URL-encoded.

Author matching is exact against the stored author string.

Some authors appear under slightly different names in different content collections.
Example: "Hailey Tran" and "Hailey M. Tran" currently behave like different author filters.

If a parameter combination has no matching content, the app will show no matching items.

Workshop excerpts currently live in the video lane as:
type=VV
catalog=2025 Vertical Workshop Excerpts

HOW TO BUILD LINKS

Link to a specific item:
https://poetryplease.org/app?item=CONTENT_ID

Link to a vertical of content:
Use one or more of:
type=...
catalog=...
author=...
book=...

Example:
https://poetryplease.org/app?author=Andrea%20Gibson&type=VV

Link to a specific item inside a vertical:
https://poetryplease.org/app?item=CONTENT_ID&author=...&type=...&catalog=...

Link to an embeddable single-item experience:
https://poetryplease.org/embed?type=...&catalog=...

Link to a specific item inside the embed:
https://poetryplease.org/embed?item=CONTENT_ID&type=...&catalog=...
